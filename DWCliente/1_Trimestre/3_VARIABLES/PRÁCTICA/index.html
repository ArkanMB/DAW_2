Veremos ahora las variables

Estas variables son:

  • Case Sensitive: es decir el lenguaje java script distingue entre mayusculas y minúsculas
  • Punto y coma tras las instrucciones: si las instrucciones son seguidas deben tener entre ellas punto y coma, si van separadas por líneas no haría falta.
  • Comentarios: se usa '/**/' para comentarios de varias líneas o // para comentarios de una sola línea
  • Variables: las variables son contenedores de datos de cualquier tipo y tienen las siguientes condiciones:
    • La longitud máxima es de 256 caracteres
    • Siempre debe empezar por una letra
    • Y puede contener el nombre: letras, números y caracter de subrayado (_)
